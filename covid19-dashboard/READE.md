# COVID-19 Data Analysis Dashboard (Power BI)

This project is a Power BI dashboard created to visualize and analyze COVID-19 data.  
The dashboard presents information about deaths and recoveries across different countries and states, helping users understand the spread and impact of COVID-19.

---

## Tools Used

- Power BI
- Power Query
- CSV Dataset

---

## Dashboard Overview

The dashboard contains multiple visualizations to explore COVID-19 data.

Main visuals included:

- Deaths by Country/Region
- Death distribution by Province/State
- Recovered cases trend over time
- Recovered cases by Province/State

These visuals help in understanding how COVID-19 affected different regions and how recovery trends changed over time.

---

## Key Features

- Interactive visualizations created using Power BI
- Time-based analysis using observation dates
- Country and state level comparison
- Clear graphical representation of COVID-19 statistics

## Dataset

The dataset contains COVID-19 statistics including:

- Country/Region
- Province/State
- Observation Date
- Confirmed Cases
- Deaths
- Recovered Cases

The dataset is used to create visualizations and analyze trends in the dashboard.

## How to View the Dashboard

1. Download the `.pbix` file from this repository.
2. Open the file using **Power BI Desktop**.
3. Explore the dashboard and interact with the visuals.
