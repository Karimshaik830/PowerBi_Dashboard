# Space Missions Analysis Dashboard (Power BI)

This project is a Power BI dashboard created to analyze and visualize space mission data. The dashboard shows mission counts, mission success rates, and launch trends over time.

---

## Tools Used
- Power BI
- Power Query
- CSV Dataset

---

## Dashboard Overview

The dashboard provides insights about space missions using multiple visualizations. It helps understand how space launches have changed over time and how different companies contribute to space exploration.

The dashboard includes the following information:

- Total number of missions
- Number of successful missions
- Number of failed missions
- Mission trends over time
- Distribution of missions by company
- Mission success vs failure analysis

---

## Key Insights

- The dataset contains **145 total space missions**.
- **118 missions were successful**.
- **28 missions failed**.
- The number of missions increased significantly in recent years.
- Different companies contribute different numbers of launches.

---

## Dashboard Visualizations

The dashboard contains the following visuals:

1. **KPI Cards**
   - Total Missions
   - Successful Missions
   - Failed Missions

2. **Line Chart**
   - Missions launched over time

3. **Bar Chart**
   - Missions by company

4. **Donut Chart**
   - Mission success vs failure distribution

## Dataset

The dataset includes information about space missions such as:

- Company
- Launch Date
- Mission Status
- Payload Mass
- Launch Site
- Vehicle Type
- Environmental conditions (temperature and wind speed)

This data was used to analyze launch patterns and mission outcomes.

---

## How to View the Dashboard

1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
3. Explore the dashboard and interact with the visuals.
