
# Power BI Dashboards

This repository contains data visualization dashboards built using Power BI.

## Tools Used
- Power BI
- Power Query
- Excel datasets

## Dashboards

### 1. COVID-19 Data Dashboard
Analyzes global COVID-19 cases, deaths, and trends using time series visualizations.

### 2. Space Missions Dashboard
Explores space mission launches by country, company, and success rate.
